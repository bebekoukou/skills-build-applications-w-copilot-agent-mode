# OctoFit Tracker

This repo contains a minimal frontend (Vite + React 19) and backend (Node + Express + TypeScript + mongoose) scaffold for the OctoFit Tracker exercise.

Ports:
- Frontend dev/preview: 5173
- Backend: 8000
- MongoDB (default): 27017

Quick start (frontend)

cd octofit-tracker/frontend
npm install
npm run dev

Quick start (backend)

cd octofit-tracker/backend
npm install
# copy .env.example to .env and adjust if needed
npm run dev
